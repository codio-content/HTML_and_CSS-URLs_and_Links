Solution : 

```html
<!DOCTYPE html>
<html>
<head>
    <title> Challenge 1 </title>
</head>
<body>
    
   <h2> Here is a nested ordered list : </h2>
    <ol>
        <li> List item 1
            <ol>
                <li>Sublist item 1</li>
                <li>Sublist item 2</li>
                <li>Sublist item 3</li>
            </ol>
        </li>
        <li> List item 2</li>
        <li> List item 3</li>
    </ol>
    
</body>
</html>
```
